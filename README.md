### テーマを選んだ理由
もともと海外旅行が好きで、これまでに14か国旅行しました。しかし、毎回頭を悩ませるのが、一緒に旅してくれる人を探すことです。
休暇が取れるタイミングや予算、行きたい地域等を友人や家族と合致させることは非常に難しいと感じています。
そこで、私と同じように、
「旅したいけど、一緒に行く人がいない、でも一人旅はハードルが高い」
という旅人同士が、旅仲間を募集できるプラットフォームがあったら便利だと考え、このテーマにしました。
